# r_launcher

Foobar is a WIP launcher for android written in flutter.

## Installation

Clone the repository and resolve dependencies.

```bash
flutter pub get
```

## Features (Implemented)
- [x] App Search and launch with animations.
- [x] Set as default launcher.
